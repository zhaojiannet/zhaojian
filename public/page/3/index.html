Windows8无法使用内置管理员账户打开xx的解决办法

发表于 2012-08-20 分类于 技术

Windows8无法使用内置管理员账户打开xx的解决办法

按win+x，选择运行，输入：”gpedit.msc”，启动本地组策略编辑器，依次选择”计算机配置”里面的”Windows设置” >> “安全设置” >> “本地策略” >> “安全选项”，在右边查找”用户帐户控制: 用于内置管理员账户的管理员批准模式”，双击打开并更改为”已启用”，确定，退出后注销重新登陆，就解决了。

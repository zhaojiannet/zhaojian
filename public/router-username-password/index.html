找回路由器上的宽带帐号和密码

发表于 2011-08-23 分类于 技术 阅读次数：

路由器上保存的宽带帐号密码忘了，打电话给网通还必须拿记住身份证去大厅。

路由安全配置较高，通常办法获取密码无效。

最终基于RASPPPOE自建一个模拟网通PPPOE的服务端，把网卡直连到WAN口上，使用cain的sniffer功能抓包成功。.
